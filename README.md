# Advent of Code 2025

Solutions written in Kotlin with no 3rd-party dependency.

## Inputs

* Inputs are not uploaded and shall be saved in `src/main/resources/day${DAY_NUM}.txt`.
  * e.g. `src/main/resources/day10.txt` for input on day10.

## Running

```shell
# For each day.
./gradlew day01
./gradlew day02
# ...

# For testing samples.
./gradlew test --info
```
